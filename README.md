# Bacciotti Contact List Plugin
Basic plugin which allows creating and editing contacts using ACF.

## Requirements / Tested on
![Wordpress 5.8.1](https://img.shields.io/badge/Wordpress-5.8.1-green)  
![PHP 7.4](https://img.shields.io/badge/PHP-7.4-blue)  


## Installation

- Download this repository
- Upload it to wp-content/plugins
- Activate it
- Use!

## Usage
- Go to "Meus Contatos" -> "Todos os Contatos" to see the contact list. In this scren you are able to edit and delete as well.
- Go to "Meus Contatos" -> "Novo Contato" to add a new one.
